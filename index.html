Introduccion

PEAK es un juego de escalada y supervivencia donde el objetivo principal es llegar a la cima de una gran montaña explorando diferentes caminos.

Objetivos del juego

El jugador debe administrar su resistencia, encontrar recursos y elegir la mejor ruta para sobrevivir al ascenso.

Mecanicas principales

Pasos basicos para escalar

  1. Explorar el terreno antes de subir.
  2. Buscar puntos seguros para escalar.
  3. Administrar la resistencia.
  4. Usar herramientas o cuerdas si es necesario.
  5. Descansar antes de continuar.

Consejos importantes

  • No escalar demasiado rápido.
  • Guardar recursos para zonas difíciles.
  • Explorar rutas alternativas.
