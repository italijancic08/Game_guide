# Guía Interactiva de PEAK

Este proyecto consiste en una página web interactiva desarrollada como guía del videojuego **PEAK**. El sitio ofrece información sobre las mecánicas del juego, los objetos más importantes y consejos para sobrevivir durante el ascenso a la montaña, incorporando funcionalidades dinámicas mediante JavaScript para mejorar la experiencia del usuario.

## Características

* Uso de estructura semántica (header, nav, main, section y footer).
* Menú de navegación con enlaces internos.
* Jerarquía correcta de títulos (h1, h2 y h3).
* Listas ordenadas y desordenadas para organizar la información.
* Tabla con los objetos importantes del juego.
* Imágenes ilustrativas.
* Formulario de contacto.
* Buscador interactivo de consejos.
* Generador de consejos aleatorios.
* Validación del formulario mediante JavaScript.

## Objetivo

El objetivo del proyecto es aplicar los conocimientos adquiridos durante la materia utilizando **HTML**, **CSS** y **JavaScript**, implementando una interfaz moderna e interactiva que permita al usuario consultar información sobre el juego PEAK de manera sencilla.

## Contenido

La página incluye:

1. Introducción al juego.
2. Explicación de las mecánicas principales.
3. Objetos importantes y su utilidad.
4. Consejos para progresar durante la partida.
5. Buscador de consejos.
6. Consejo aleatorio.
7. Formulario de contacto con validaciones.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (ES6)

## Funcionalidades implementadas

* Archivo JavaScript externo (`script.js`).
* Uso de funciones para organizar la lógica del programa.
* Implementación de múltiples eventos (`click`, `input`, `submit` y `mouseover`).
* Uso de un array para almacenar los consejos del juego.
* Manipulación dinámica del DOM mediante creación y modificación de elementos HTML.
* Buscador de consejos en tiempo real.
* Generación de consejos aleatorios.
* Validación de campos obligatorios del formulario.
* Manejo de errores utilizando `try...catch`.

## Mejoras visuales

* Implementación de una paleta de colores mediante variables CSS (`:root`).
* Tipografía personalizada utilizando Google Fonts.
* Diseño moderno con sombras, bordes redondeados y efectos de transición.
* Tarjetas de contenido con animaciones al pasar el cursor.
* Botones personalizados con efectos visuales.
* Uso de Flexbox para el menú de navegación.
* Uso de CSS Grid para organizar el contenido principal.
* Diseño responsive mediante Media Queries para computadoras, tablets y dispositivos móviles.
