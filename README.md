# Guia HTML de PEAK
Este proyecto consiste en una página web desarrollada en HTML que presenta una guía básica del videojuego PEAK.

## Caracteristicas
- Uso de estructura semántica (header, nav, main, section, footer).
- Menú de navegación con enlaces internos.
- Jerarquía correcta de títulos (h1, h2, h3)
- Listas ordenadas y desordenadas para organizar información
- Tabla de datos con objetos del juego
- Imágenes
- Formulario de contacto

## Objetivo
El objetivo del proyecto es aplicar conceptos fundamentales de desarrollo web con HTML, como la estructura semántica, la navegación interna y la organización del contenido en una página web.

## Contenido
La pagina incluye:
1. Introduccion al juego
2. Explicación de las mecánicas principales
3. Lista de objetos importantes
4. Consejos y pasos para progresar en el juego
5. Formulario de contacto

## Tecnologias utilizadas
- HTML5
